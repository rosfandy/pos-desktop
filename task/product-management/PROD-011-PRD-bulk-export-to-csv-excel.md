# [PROD-011] Bulk Export to CSV/Excel

**Module**: Product Management
**Priority**: P1
**Complexity**: S
**Spec Reference**: product-management-spec.md Section 4

## Description
Export daftar produk ke file CSV atau Excel dengan filter yang sama seperti list view.

## Acceptance Criteria
- [ ] Export sesuai filter aktif (search, category)
- [ ] Format file .xlsx atau .csv
- [ ] File bisa dibuka di Excel
- [ ] Filename mengandung tanggal
