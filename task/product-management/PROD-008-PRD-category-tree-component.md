# [PROD-008] CategoryTree Component

**Module**: Product Management
**Priority**: P0
**Complexity**: S
**Spec Reference**: product-management-spec.md Section 7

## Description
Komponen tree view untuk menampilkan dan mengelola kategori. Bisa expand/collapse sub-kategori. Drag-and-drop untuk ubah parent (optional, bisa jadi P1).

## Acceptance Criteria
- [ ] Menampilkan kategori dalam hierarki tree
- [ ] Expand/collapse sub-kategori
- [ ] Add new category dari tree
- [ ] Edit nama kategori inline atau modal
- [ ] Delete kategori dengan konfirmasi
