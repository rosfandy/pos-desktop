# [PROD-006] ProductList + ProductCard Components

**Module**: Product Management
**Priority**: P0
**Complexity**: M
**Spec Reference**: product-management-spec.md Section 7

## Description
Halaman daftar produk dengan tabel atau grid view. ProductCard untuk grid view. Kolom: nama, SKU, barcode, kategori, harga jual, stok, status. Toggle active/inactive.

## Acceptance Criteria
- [ ] Tabel menampilkan produk dengan pagination
- [ ] Grid view alternatif dengan ProductCard
- [ ] Search/filter by name/SKU real-time
- [ ] Filter by category dropdown
- [ ] Badge "Stok Hampir Habis" untuk produk low stock
- [ ] Tombol edit dan delete pada setiap row/card
