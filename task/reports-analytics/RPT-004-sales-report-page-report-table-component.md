# [RPT-004] SalesReportPage + ReportTable Component

**Module**: Reports & Analytics
**Priority**: P0
**Complexity**: M
**Spec Reference**: reports-analytics-spec.md Section 7

## Description
Halaman laporan penjualan dengan tabel detail. Filter: date range, group by (date, product, category, cashier). Sortable columns. Pagination.

## Acceptance Criteria
- [ ] Tabel menampilkan transaksi/detail penjualan
- [ ] Filter date range dengan date picker
- [ ] Group by toggle: tanggal / produk / kategori / kasir
- [ ] Sort by kolom (tanggal, total, qty)
- [ ] Pagination 50 rows per page
- [ ] Summary row: total revenue, total transactions
