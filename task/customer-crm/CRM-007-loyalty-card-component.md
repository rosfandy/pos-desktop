# [CRM-007] LoyaltyCard Component

**Module**: Customer CRM
**Priority**: P0
**Complexity**: S
**Spec Reference**: customer-crm-spec.md Section 7

## Description
Kartu informasi membership pelanggan. Menampilkan: tier badge, points balance, total spent, progress bar ke tier berikutnya.

## Acceptance Criteria
- [ ] Tier badge dengan icon/warna sesuai tier
- [ ] Points balance besar dan terbaca
- [ ] Progress bar dengan label "Rp X lagi ke [Next Tier]"
- [ ] Total spent terformat rupiah
- [ ] Responsive card layout
