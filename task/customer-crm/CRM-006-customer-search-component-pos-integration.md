# [CRM-006] CustomerSearch Component (POS Integration Ready)

**Module**: Customer CRM
**Priority**: P0
**Complexity**: S
**Spec Reference**: customer-crm-spec.md Section 7

## Description
Komponen search pelanggan yang bisa digunakan di POS Terminal. Quick search by phone dengan autocomplete dropdown. Menampilkan poin dan tier di hasil.

## Acceptance Criteria
- [ ] Input phone dengan autocomplete
- [ ] Dropdown hasil menampilkan nama, tier, poin
- [ ] Bisa select customer untuk transaksi
- [ ] Bisa create new customer dari hasil pencarian jika tidak ditemukan
- [ ] Component reusable di POS dan CustomersPage
