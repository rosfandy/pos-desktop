# [CRM-011] Credit Sales / Hutang Piutang Tracking

**Module**: Customer CRM
**Priority**: P2
**Complexity**: M
**Spec Reference**: PRD Section 5.4

## Description
Fitur mencatat penjualan kredit. Tabel hutang dengan status lunas/belum. Pembayaran cicilan.

## Acceptance Criteria
- [ ] Tandai transaksi sebagai kredit (paymentMethod: 'credit')
- [ ] Tampilkan daftar hutang per pelanggan
- [ ] Input pembayaran cicilan
- [ ] Status hutang: lunas / belum lunas
