# [SHIFT-007] ShiftPage: History & List

**Module**: Shift Management
**Priority**: P0
**Complexity**: S
**Spec Reference**: shift-management-spec.md Section 2

## Description
Halaman riwayat shift dengan tabel. Kolom: tanggal, kasir, modal awal, penjualan, selisih, status. Filter by date range. Sort by date.

## Acceptance Criteria
- [ ] Tabel menampilkan shift history dengan pagination
- [ ] Filter by date range
- [ ] Filter by kasir (dropdown)
- [ ] Sort by date descending default
- [ ] Klik row untuk detail shift (summary)
