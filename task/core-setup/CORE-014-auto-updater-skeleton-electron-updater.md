# [CORE-014] Auto-Updater Skeleton (electron-updater)

**Module**: Core & Setup
**Priority**: P2
**Complexity**: M
**Spec Reference**: PRD Section 5.7

## Description
Setup electron-updater dengan event listeners untuk check update, download progress, install. UI notification untuk update available. Dummy update server config.

## Acceptance Criteria
- [ ] App check update saat startup
- [ ] Notifikasi muncul jika ada update baru
- [ ] Bisa trigger install update dari UI
- [ ] Log update events ke file
