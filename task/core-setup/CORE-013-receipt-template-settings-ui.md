# [CORE-013] Receipt Template Settings UI

**Module**: Core & Setup
**Priority**: P1
**Complexity**: S

## Description

## Acceptance Criteria
- [ ] Form bisa edit header dan footer text struk
- [ ] Checkbox untuk show/hide field opsional
- [ ] Preview struk di sidebar sebelum di-save
- [ ] Settings tersimpan ke tabel settings
