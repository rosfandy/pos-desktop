# [CORE-002] Setup Tailwind CSS + shadcn/ui Base Components

**Module**: Core & Setup
**Priority**: P0
**Complexity**: S
**Spec Reference**: core-setup-spec.md Section 2

## Description
Instal dan konfigurasi Tailwind CSS dengan tema POS (warna primary, spacing touch-friendly). Setup shadcn/ui dengan komponen dasar: Button, Input, Dialog, Table, Card.

## Acceptance Criteria
- [ ] Tailwind config dengan custom theme tokens
- [ ] shadcn/ui CLI berfungsi untuk menambah komponen
- [ ] Komponen Button, Input, Dialog, Table, Card tersedia di `src/components/ui/`
- [ ] Dark mode class strategy siap digunakan
